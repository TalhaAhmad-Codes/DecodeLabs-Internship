# Project 1: Responsive Web design using Vanilla HTML/CSS/JS

---

## Description

This project demonstrates responsive web design by using only HTML, CSS, and JS and no framework. It's a simple webpage having:
- A header and a footer.
- A div block containing description on left side and an image on right.
- At bottom, there's also a div containing three cards with some animations.

## How it Run

There're to two ways to run it, which are:

- **Way 1:** Locate the `.html` file and open it with your browser.
- **Way 2:** This is deployed on vercel so follow this link [*Click Here*](https://decode-labs-proj1.vercel.app/)
